1. Diagnóstico geral

A ideia central é boa: investigar se a rede de proteção do DF está orientando suas prioridades reais para os fatores que efetivamente expõem crianças e adolescentes à morte violenta. O melhor caminho é transformar a pesquisa em uma comparação entre:

a realidade empírica das mortes

versus

as prioridades institucionais declaradas ou praticadas

versus

o conteúdo jurídico da prioridade absoluta.

A prioridade absoluta tem base constitucional no art. 227 da Constituição e no art. 4º do ECA, e envolve não apenas preferência retórica, mas dever de proteção reforçada, inclusive quanto ao direito à vida e à proteção contra violência, negligência, crueldade e opressão. [Planalto]

3. Versão mais crítica e mais próxima da necropolítica

:::writing{variant="document" id="90314"}

Hipótese crítica

A hipótese é que existe um descompasso entre a prioridade absoluta como promessa constitucional de proteção integral e sua mobilização jurídica e institucional concreta. Embora o direito afirme que crianças e adolescentes devem receber proteção prioritária contra toda forma de violência, a categoria da prioridade absoluta tende a tornar juridicamente mais visíveis violações situadas no âmbito familiar, privado e interpessoal, enquanto fatores necropolíticos de produção da morte — racismo estrutural, violência estatal, guerra às drogas, desigualdade territorial, precarização do cuidado, adultocentrismo, cisheteropatriarcado e exposição de comunidades inteiras à morte — permanecem marginalizados. Assim, a prioridade absoluta pode funcionar menos como tecnologia de prevenção da letalidade infantojuvenil e mais como categoria jurídica de tutela seletiva de conflitos considerados legíveis pelo direito.

Essa é a formulação mais potente teoricamente.

    """
    Transforma 'quilombo território' em 'quilombo_territorio'.

    texto = unicodedata.normalize("NFKD", texto)

def encontrar_paginas_html(pasta, filename):
    """
    Encontra arquivos no padrão:
    {filename}_pagina1.html
    {filename}_pagina2.html
    etc.
    """
    if not pasta.exists():
        raise FileNotFoundError(f"A pasta não existe: {pasta}")

    padrao = re.compile(rf"^{re.escape(filename)}_pagina(\d+)\.html$", re.IGNORECASE)

    arquivos = []

    for arquivo in pasta.iterdir():
        if arquivo.is_file():
            match = padrao.match(arquivo.name)
            if match:
                numero_pagina = int(match.group(1))
                arquivos.append((numero_pagina, arquivo))

    arquivos.sort(key=lambda x: x[0])

    return [arquivo for _, arquivo in arquivos]

refaz o código então com estas alterações

Segue o código refeito, já com estas alterações:

  • lê todos os arquivos da pasta Carlos;
  • pega apenas os arquivos no padrão {filename}_pagina{n}.html;
  • exemplo: quilombo_territorio_pagina1.html, quilombo_territorio_pagina2.html;
  • extrai o const itens = [...];
  • extrai o texto de cada <p class="mb-0 small">;
  • remove tags internas como <b>, <i>, <span>;
  • insere esse texto no campo abstract;
  • salva no banco {pesquisa}.db, por exemplo quilombos.db;
  • cria/atualiza a tabela com o nome de filename, por exemplo quilombo_territorio.

from bs4 import BeautifulSoup
import re
import demjson3
import sqlite3
import os
import unicodedata
from pathlib import Path


# =========================
# CONFIGURAÇÕES DA PESQUISA
# =========================

pesquisa = "quilombos"
termo = "quilombo território"


def slugify(texto):
    """
    Transforma 'quilombo território' em 'quilombo_territorio'.
    Remove acentos e caracteres problemáticos para nome de arquivo/tabela.
    """
    texto = unicodedata.normalize("NFKD", texto)
    texto = texto.encode("ascii", "ignore").decode("ascii")
    texto = texto.lower()
    texto = re.sub(r"[^a-z0-9]+", "_", texto)
    texto = texto.strip("_")
    return texto


filename = slugify(termo)

try:
    SCRIPT_DIR = Path(__file__).resolve().parent
except NameError:
    SCRIPT_DIR = Path.cwd()

PASTA_HTML = SCRIPT_DIR / "Carlos"
DB_PATH = SCRIPT_DIR / f"{pesquisa}.db"


# =========================
# FUNÇÕES AUXILIARES
# =========================

def quote_identifier(nome):
    """
    Protege nomes de tabela no SQLite.
    """
    return '"' + nome.replace('"', '""') + '"'


def limpar_texto_html(elemento):
    """
    Remove tags HTML internas e normaliza espaços.

    Exemplo:
    terreiro-<b>quilombo</b> -> terreiro-quilombo
    Neste <b>território</b> -> Neste território
    """
    if elemento is None:
        return None

    texto = elemento.get_text("", strip=False)
    texto = re.sub(r"\s+", " ", texto)
    texto = texto.strip()

    return texto


def encontrar_paginas_html(pasta, filename):
    """
    Encontra arquivos no padrão:
    {filename}_pagina1.html
    {filename}_pagina2.html
    {filename}_pagina3.html
    etc.

    Exemplo:
    quilombo_territorio_pagina1.html
    quilombo_territorio_pagina2.html
    """
    if not pasta.exists():
        raise FileNotFoundError(f"A pasta não existe: {pasta}")

    padrao = re.compile(
        rf"^{re.escape(filename)}_pagina(\d+)\.html$",
        re.IGNORECASE
    )

    arquivos = []

    for arquivo in pasta.iterdir():
        if arquivo.is_file():
            match = padrao.match(arquivo.name)

            if match:
                numero_pagina = int(match.group(1))
                arquivos.append((numero_pagina, arquivo))

    arquivos.sort(key=lambda x: x[0])

    return [arquivo for _, arquivo in arquivos]


def encontrar_script_com_itens(soup):
    """
    Procura o script que contém 'const itens'.
    """
    for script in soup.find_all("script"):
        conteudo = script.string or script.get_text()

        if conteudo and re.search(r"const\s+itens", conteudo):
            return conteudo

    return None


def extrair_array_itens(conteudo_script):
    """
    Extrai o array JavaScript declarado como:
    const itens = [...]
    """
    match = re.search(
        r"const\s+itens\s*=\s*(
%%MB0%%
);",
        conteudo_script,
        re.DOTALL
    )

    if not match:
        return []

    try:
        return demjson3.decode(match.group(1))
    except Exception as e:
        print(f"Erro ao converter const itens para objeto Python: {e}")
        return []


def extrair_itens_do_html(caminho_html):
    """
    Extrai, de cada página HTML:
    1. os dados do array JS const itens = [...]
    2. o texto interno de cada <p class="mb-0 small">

    O texto dos parágrafos é inserido em item["concept"],
    que depois será gravado no campo abstract do banco SQLite.
    """
    with open(caminho_html, "r", encoding="utf-8") as f:
        html = f.read()

    soup = BeautifulSoup(html, "html.parser")

    conteudo_script = encontrar_script_com_itens(soup)

    if not conteudo_script:
        print(f"'const itens' não encontrado em: {caminho_html.name}")
        return []

    itens = extrair_array_itens(conteudo_script)

    if not itens:
        print(f"Nenhum item extraído em: {caminho_html.name}")
        return []

    paragrafos = soup.select("p.mb-0.small")

    textos = [
        limpar_texto_html(p)
        for p in paragrafos
    ]

    if len(textos) != len(itens):
        print(
            f"Atenção em {caminho_html.name}: "
            f"{len(itens)} itens encontrados, mas "
            f"{len(textos)} textos em <p class='mb-0 small'>."
        )

    for i, item in enumerate(itens):
        if i < len(textos):
            item["concept"] = textos[i]
        else:
            item["concept"] = None

    print(f"{caminho_html.name}: {len(itens)} itens extraídos.")

    return itens


def remover_duplicatas_por_id(itens):
    """
    Remove duplicatas mantendo a primeira ocorrência de cada id.
    """
    vistos = set()
    unicos = []

    for item in itens:
        id_art = item.get("id")

        if not id_art:
            continue

        if id_art not in vistos:
            vistos.add(id_art)
            unicos.append(item)

    return unicos


def salvar_no_sqlite(itens, db_path, nome_tabela):
    """
    Salva os registros no banco SQLite.
    O banco terá o nome definido em pesquisa, por exemplo:
    quilombos.db

    A tabela terá o nome definido em filename, por exemplo:
    quilombo_territorio
    """
    tabela = quote_identifier(nome_tabela)

    conn = sqlite3.connect(db_path, isolation_level=None)

    conn.execute(f"""
        CREATE TABLE IF NOT EXISTS {tabela} (
            type TEXT,
            id TEXT PRIMARY KEY,
            title TEXT,
            autor TEXT,
            resource_title TEXT,
            publish_year TEXT,
            volume TEXT,
            issue TEXT,
            first_page TEXT,
            last_page TEXT,
            doi TEXT,
            abstract TEXT
        ) STRICT
    """)

    for item in itens:
        conn.execute(f"""
            INSERT OR REPLACE INTO {tabela}
            (
                type,
                id,
                title,
                autor,
                resource_title,
                publish_year,
                volume,
                issue,
                first_page,
                last_page,
                doi,
                abstract
            )
            VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
        """, (
            item.get("type"),
            item.get("id"),
            item.get("title"),
            item.get("autor"),
            item.get("resource_title"),
            item.get("publish_year"),
            item.get("volume"),
            item.get("issue"),
            item.get("first_page"),
            item.get("last_page"),
            item.get("doi"),
            item.get("concept"),
        ))

    conn.close()

arquivos_html = encontrar_paginas_html(PASTA_HTML, filename)

if not arquivos_html:
    raise FileNotFoundError(
        f"Nenhum arquivo encontrado em {PASTA_HTML} com o padrão "
        f"{filename}_pagina{{numero}}.html"
    )

print(f"Arquivos encontrados: {len(arquivos_html)}")

todos_itens = []

for caminho_html in arquivos_html:
    itens_pagina = extrair_itens_do_html(caminho_html)
    todos_itens.extend(itens_pagina)

json_obj = remover_duplicatas_por_id(todos_itens)

if not json_obj:
    raise RuntimeError("Nenhum artigo extraído dos HTMLs encontrados.")

print(f"Total de artigos únicos: {len(json_obj)}")

salvar_no_sqlite(json_obj, DB_PATH, filename)

print("Concluído.")
print(f"Banco salvo em: {DB_PATH}")
print(f"Tabela criada/atualizada: {filename}")

como faço para exportar esta conversa

Você tem três caminhos, dependendo do que quer “exportar”:

1. Exportar todos os seus dados/conversas do ChatGPT

No ChatGPT web:

  1. Clique no seu perfil/nome, no canto inferior esquerdo.
  2. Vá em Settings / Configurações.
  3. Abra Data Controls / Controles de dados.
  4. Clique em Export Data / Exportar dados.
  5. Confirme a solicitação.
  6. Você receberá um e-mail com um link para baixar o arquivo exportado.

A exportação inclui seu histórico de conversas e outros dados da conta; em exports maiores, as conversas podem vir em arquivos como conversations.json ou arquivos numerados. [OpenAI Help Center]

2. Compartilhar só esta conversa por link

Na conversa aberta:

  1. Clique no botão de compartilhar no topo da conversa.
  2. Gere um link compartilhável.
  3. Copie o link.

Segundo a central da OpenAI, links compartilhados criam uma URL única para aquela conversa e podem ser gerados pelo site ou pelos apps. [OpenAI Help Center]

3. Salvar esta conversa em PDF

No navegador:

  1. Pressione Ctrl + P.
  2. Em “Destino”, escolha Salvar como PDF.
  3. Ajuste margens/escala se necessário.
  4. Salve o arquivo.

Para seu caso, como esta conversa tem material de pesquisa e código, eu recomendo Ctrl + P → Salvar como PDF para preservar a leitura humana, e também Export Data se quiser arquivar tudo em formato bruto.
